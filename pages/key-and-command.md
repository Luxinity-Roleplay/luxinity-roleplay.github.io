# Key's and Command's

Hallo semua berikut adalah key dan command yang berlaku di server kami :

## Keys

Dibawah ini adalah beberapa fungsi keygen di server kami :

| KEY   | FUNGSI                               |
| ----- | ------------------------------------ |
| ENTER | Masuk/Keluar Interior                |
| Y     | Menyalakan/Mematikan mesin kendaraan |
