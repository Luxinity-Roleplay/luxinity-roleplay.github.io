## Install SA:MP  

- Pertama pastikan kalian sudah memiliki atau sudah terinstall game GTA San Andreas di PC/Laptop kalian masing-masing. Jika kalian belum memiliki game GTA San Andreas silahkan kalian beli atau [download disini](https://drive.google.com/drive/folders/1QKFfPkNyd13FZ_aFWFL40SmS8my1yfsR?usp=sharing)
- Jika sudah memiliki game GTA SAnya silahkan kalian install SAMP melalui link di atas
- Setelah selesai mendownload SAMP, silahkan install seperti biasa, setelah muncul ini :

![tutorial install samp client](https://files.catbox.moe/tidmmo.png)

Silahkan anda klik "Browse" dan arahkan pada folder GTASA yang telah anda install sebelumnya. Misal : *D:/GAMES/GTA SA*

- Setelah install selesai, silahkan pergi ke folder GTA SA milik kalian lalu jalankan "samp.exe". lalu nanti menu SA:MP akan muncul.

![tutorial install samp client](https://files.catbox.moe/d252n5.png)

**Keterangan:**

- Nomor 1: Tempat menuliskan nickname.
- Nomor 2: Tombol untuk menghapus suatu server dari list Favorites.
- Nomor 3: Tombol untuk menambahkan server baru di list Favorites.
- Nomor 4: Tombol "play" berguna untuk memulai game setelah memilih server.
